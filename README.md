instrucciones provisionales para el uso de esta aplicacion:
1. crear una carpeta nueva
2. descargar el archivo a traves de la terminal git bash con el comando git clone
3. abir el programa desde su entorno de desarrollo habitual
4. ubicarse en la ruta: final/src/main/java/com/example/demo
5. abrir el archivo FinalApplication.java
6. correr el archivo
7. abrir postman
8. abrir el archivo rutasPostman.txt
9. ejecutar las rutas segun lo que necesite
10. en caso de tener problemas abrir la aplicacion docker o correr el contenedor de postgres correspondiente
